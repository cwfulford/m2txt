NAME

m2txt - creates a mail message to send to a mail to sms text gateway.

SYNOPSIS

m2txt [-c <config_file>] [-d] [-l] [-r <rcpt_email_addr>] [-v] <clientid> .br

AVAILABILITY

m2txt is a Bourne shell script which should work equally well on all versions of UNIX, Linux and Mac OS X.

DESCRIPTION

m2txt uses the event and date/time parameters received on the command line to create a subject line to send to the sms gateway. A brief message is also appended to prevent problems which may be experienced if mailx receives a null message.

m2txt is designed to be called by the orage calendar programme to text alerts to the calendar user but could also be used from the command line.

OPTIONS

-c Use the specified configuration file in preference to the default "/usr/local/etc/m2txt.cf". If no configuration file is found the script will attempt to soldier on with the information it has from the command line.

-d Start in debug mode. The -x option is set within the shell.

-v Set verbose mode. Ordinarily m2txt operates silently unless show stopping problems are detected. In verbose mode m2txt reports on each significant action. -V m2txt reports the release details and terminates.

EXAMPLES

m2txt  <start-time> <event> <recipient>

In the Orage calendar the syntax for setting the Alarm is "/usr/local/m2txt.cf <&ST> <&T>".

BUGS

The script is quite crude having little or no error checking. The consequences of unexpected input are untested.
